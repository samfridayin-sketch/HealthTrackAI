# HealthTrack-AI 🏥

AI powered chatbot for weight loss tracking and personalized nutrition guidance.

## Features
- 🤖 AI nutrition chatbot
- 📊 Calorie tracking
- 🔥 Daily streaks
- 💪 Personalized weight loss guidance
- 📈 Weekly progress predictions

## Tech Stack
- Backend: Python + FastAPI
- Frontend: React
- AI: Claude API (Anthropic)
- Database: Firebase
- Deployment: Docker + Vercel

## Getting Started

1. Clone the repo

\```
git clone https://github.com/samfridayin-sketch/HealthTrack-AI.git
\```

2. Set up environment

\```
cp .env.example .env
\```

3. Run with Docker

\```
docker-compose up
\```
